==== CONTAINS ====
Text version of result: ./bachelorsThesis.pdf
Tex sources are located in: ./latex/
source folder is: ./sources/
program is located in: ./binary/

==== RUN ====
To compile program go to sources and run "$ant" this will compile and run program.

For correct run of this project you need two libraries, they are in ./sources/lib/

==== JAVADOC ====
Javadoc is located in ./documentation/

To build egain javadoc go to sourcesand run "$ant doc"
